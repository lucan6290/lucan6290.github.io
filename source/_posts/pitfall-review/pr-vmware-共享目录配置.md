## 什么是共享目录

假设现在 Windows 有一个文件叫 `test.txt`，我现在需要将这个文件放在 Linux 下使用，怎么办？

### 方法一：直接拷贝

直接把文件拖到 Linux（虚拟机）下使用。

> ❌ 不推荐，因为这样等价于拷贝一份新的数据给 Linux，占用空间就会翻倍。

### 方法二：配置共享目录

配置一个共享目录，让 Linux 直接去访问 Windows 的文件。

> ✅ 推荐，节省空间且文件实时同步。

## 配置共享目录步骤

### 1. 在 Windows 中创建共享目录

先在 Windows 中创建一个新的目录，作为共享目录。

### 2. 在虚拟机中配置共享文件夹

在虚拟机（Linux / Windows 系统）下，操作如下步骤：

1. 打开虚拟机设置
2. 选择「选项」→「共享文件夹」
3. 选择「总是启用」
4. 添加之前创建的 Windows 目录

### 3. 在 Linux 中访问共享目录

在终端中切换到共享目录下：

```bash
cd /mnt/hgfs
ls
```

如果输入 `ls`（Linux 命令：查看当前路径下的所有文件），可以看到你自己设置的共享文件夹，那么说明共享成功了。

## 常见问题

### 看不到共享文件夹怎么办？

如果输入 `ls`，没有看到任何东西，只需要：

1. 重启一下虚拟机
2. 再输入以下命令：

```bash
cd /mnt/hgfs
ls
```

即可看到共享文件夹。
