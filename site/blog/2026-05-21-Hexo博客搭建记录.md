## 前言

本文记录我搭建「箓川码笺」博客的完整过程，希望能帮助到想要搭建个人博客的朋友们。

---

## 一、环境准备

### 1.1 安装 Node.js

```bash
# 推荐使用 nvm 管理 Node 版本
nvm install 18
nvm use 18
```

### 1.2 安装 Hexo

```bash
npm install -g hexo-cli
```

---

## 二、初始化博客

### 2.1 创建项目

```bash
hexo init blog
cd blog
npm install
```

### 2.2 基础配置

修改 `_config.yml`：

```yaml
title: 箓川码笺
author: lucan
language: zh-CN
timezone: 'Asia/Shanghai'
```

---

## 三、安装 Butterfly 主题

### 3.1 下载主题

```bash
npm install hexo-theme-butterfly
```

### 3.2 配置主题

修改 `_config.yml`：

```yaml
theme: butterfly
```

创建 `_config.butterfly.yml` 进行主题定制。

---

## 四、部署到 GitHub Pages

### 4.1 配置部署

```yaml
deploy:
  type: git
  repo: https://github.com/username/username.github.io.git
  branch: main
```

### 4.2 安装部署插件

```bash
npm install hexo-deployer-git
```

---

## 五、定制命令

为了自动完成文件命名、目录创建和 Front Matter 生成，定制了 `hexo np` 命令。

**命令格式：**
```bash
hexo np <一级前缀> <二级前缀> <标题>
```

**使用示例：**
```bash
hexo np ts vue3 "Vue3组合式 API详解"
# → source/_posts/tech-study/ts-vue3-Vue3组合式 API详解.md
```

> 💡 详细说明请参考 [定制创建文章命令记录](/blog/定制创建文章命令记录)

---

## 六、常用命令

```bash
hexo clean     # 清理缓存
hexo generate  # 生成静态文件
hexo server    # 本地预览
hexo deploy    # 部署到远程
hexo np        # 创建新文章（自定义命令）
```

---

## 七、总结

Hexo + Butterfly 是一个轻量、美观、易扩展的博客方案。通过定制 `hexo np` 命令，可以自动完成文章创建的繁琐工作，专注于内容创作即可。
