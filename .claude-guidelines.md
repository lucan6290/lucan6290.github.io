# 箓川码笺 —— 项目开发规范

> 本文档供 Claude Code 等 AI 助手阅读，了解本项目的规范和约定。

## 项目概览

- **项目名称**：箓川码笺
- **项目类型**：Hexo 静态博客
- **技术栈**：Hexo 8.1.2 + Butterfly 5.5.5-b1
- **部署**：GitHub Pages + GitHub Actions
- **作者**：lucan
- **语言**：中文为主，技术术语可用英文

## 核心规范

### 1. 文章文件命名规范

遵循 `[分类前缀]-[具体主题].md` 格式：

| 分类 | 前缀 | 示例 |
|------|------|------|
| 技术研习 | `ai-` / `fe-` / `be-` / `algo-` | `ai-agent-入门笔记.md` |
| 踩坑复盘 | `env-` / `debug-` | `env-nodejs-版本管理.md` |
| 项目实战 | `proj-` | `proj-blog-搭建记录.md` |
| 成长随笔 | `growth-` | `growth-求职经历总结.md` |
| 资源分享 | `tool-` / `resource-` | `tool-hexo-博客搭建.md` |

### 2. 目录结构

```
blog/
├── source/
│   ├── _posts/          # 博客文章（按分类文件夹组织）
│   │   ├── tech-study/  # 技术研习
│   │   ├── pitfall-review/  # 踩坑复盘
│   │   ├── project-practice/  # 项目实战
│   │   ├── growth-essay/  # 成长随笔
│   │   └── resource-sharing/  # 资源分享
│   ├── _drafts/         # 草稿
│   ├── about/           # 关于页面
│   ├── categories/      # 分类页
│   ├── tags/            # 标签页
│   └── img/             # 图片资源
│       ├── covers/      # 分类封面图
│       └── posts/       # 文章图片
├── themes/
│   └── butterfly/       # Butterfly 主题（不提交到 git）
├── scaffolds/           # 文章模板（按分类）
├── scripts/             # Hexo 自定义脚本
├── _config.yml          # Hexo 主配置
├── _config.butterfly.yml # Butterfly 主题配置
└── source/css/custom.css # 自定义样式
```

### 3. 文章 Front Matter

每篇文章必须包含以下内容：

```yaml
---
title: 文章标题
date: 2026-05-22 15:30:00
categories:
  - [一级分类, 二级分类]
tags: [标签1, 标签2, 标签3]
description: 文章简介（100-200字）
cover: /img/covers/封面.svg  # 按分类自动设置
---
```

### 4. 写作规范

- **语言**：中文为主，清晰易懂
- **格式**：Markdown，合理使用标题、列表、表格、代码块
- **代码块**：注明语言，示例要正确可运行
- **图片**：放在 `source/img/posts/` 或文章同名文件夹

### 5. 协作规则

#### 必须遵守

1. ✅ 不要直接修改 `themes/butterfly/` 下的文件
2. ✅ 主题定制在 `_config.butterfly.yml` 和 `source/css/custom.css`
3. ✅ 重大变更先确认再执行
4. ✅ 新增文章后，运行 `npm run build` 验证构建

#### 推荐做法

- 先了解现有文章的风格再创建新文章
- 复杂问题拆分成小步骤，逐步执行

## 常用命令

| 命令 | 说明 |
|------|------|
| `npm run server` | 启动本地预览（http://localhost:4000） |
| `npm run build` | 生成静态文件到 `public/` |
| `npm run clean` | 清理缓存和 `public/` |
| `npx hexo new "文章标题"` | 创建新文章 |

## Git 与部署

- 推送到 `main` 分支自动触发 GitHub Actions 部署
- 也可手动触发：https://github.com/lucan6290/lucan6290.github.io/actions
- Commit 信息使用简洁中文描述

## 主题安装

如果 `themes/butterfly/` 目录不存在：

1. 访问 https://github.com/jerryc127/hexo-theme-butterfly/releases/tag/v5.5.5-b1
2. 下载 Source code (zip)
3. 解压到 `themes/butterfly/`

详见 `THEME_SETUP.md`
