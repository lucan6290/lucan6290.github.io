## 前言

Vim 是 Jetson 开发板上最常用的文本编辑器之一。本文整理了 Vim 的常用命令，方便在远程 SSH 编辑配置文件时快速查阅。

---

## 一、模式切换

| 命令 | 功能 |
|------|------|
| `i` | 进入插入模式（光标前） |
| `a` | 进入插入模式（光标后） |
| `I` | 进入插入模式（行首） |
| `A` | 进入插入模式（行尾） |
| `o` | 在下方新建行并进入插入模式 |
| `O` | 在上方新建行并进入插入模式 |
| `v` | 进入可视模式 |
| `V` | 进入行可视模式 |
| `Ctrl+v` | 进入块可视模式 |
| `Esc` 或 `Ctrl+[` | 返回普通模式 |
| `:` | 进入命令模式 |

---

## 二、光标移动

### 基本移动

| 命令 | 功能 |
|------|------|
| `h` `j` `k` `l` | 左/下/上/右移动 |
| `w` | 移动到下一个单词开头 |
| `b` | 移动到上一个单词开头 |
| `e` | 移动到单词结尾 |
| `0` | 移动到行首 |
| `^` | 移动到行首非空字符 |
| `$` | 移动到行尾 |
| `gg` | 移动到文件开头 |
| `G` | 移动到文件末尾 |
| `:[行号]` | 跳转到指定行号 |

### 屏幕滚动

| 命令 | 功能 |
|------|------|
| `Ctrl+f` | 向下翻页 |
| `Ctrl+b` | 向上翻页 |
| `Ctrl+d` | 向下半页 |
| `Ctrl+u` | 向上半页 |
| `zz` | 当前行居中 |
| `zt` | 当前行置顶 |
| `zb` | 当前行置底 |

---

## 三、删除操作

| 命令 | 功能 |
|------|------|
| `x` | 删除当前字符 |
| `X` | 删除前一个字符 |
| `dw` | 删除单词 |
| `dd` | 删除当前行 |
| `D` / `d$` | 删除到行尾 |
| `d0` | 删除到行首 |
| `dgg` | 删除到文件开头 |
| `dG` | 删除到文件末尾 |
| `:[起始行],[结束行]d` | 删除指定范围行 |

---

## 四、复制粘贴

| 命令 | 功能 |
|------|------|
| `yy` / `Y` | 复制当前行 |
| `yw` | 复制单词 |
| `p` | 在光标后粘贴 |
| `P` | 在光标前粘贴 |
| `"*p` | 从系统剪贴板粘贴 |
| `"*y` | 复制到系统剪贴板 |

---

## 五、搜索替换

### 搜索

| 命令 | 功能 |
|------|------|
| `/关键词` | 向前搜索 |
| `?关键词` | 向后搜索 |
| `n` | 下一个匹配项 |
| `N` | 上一个匹配项 |
| `*` | 搜索当前单词（向前） |
| `#` | 搜索当前单词（向后） |

### 替换

| 命令 | 功能 |
|------|------|
| `:s/旧/新/` | 替换当前行第一个匹配 |
| `:s/旧/新/g` | 替换当前行所有匹配 |
| `:%s/旧/新/g` | 替换全文所有匹配 |
| `:%s/旧/新/gc` | 替换全文（带确认） |

---

## 六、文件操作

| 命令 | 功能 |
|------|------|
| `:w` | 保存文件 |
| `:w 文件名` | 另存为 |
| `:q` | 退出 |
| `:q!` | 强制退出（不保存） |
| `:wq` 或 `:x` | 保存并退出 |
| `:e 文件名` | 打开文件 |

---

## 七、窗口管理

| 命令 | 功能 |
|------|------|
| `:sp [文件]` | 水平分割窗口 |
| `:vsp [文件]` | 垂直分割窗口 |
| `Ctrl+w h/j/k/l` | 切换窗口 |
| `Ctrl+w =` | 等分窗口大小 |
| `Ctrl+w q` | 关闭当前窗口 |

---

## 八、高级功能

### 撤销重做

| 命令 | 功能 |
|------|------|
| `u` | 撤销 |
| `Ctrl+r` | 重做 |
| `U` | 撤销整行修改 |

### 宏录制

| 命令 | 功能 |
|------|------|
| `q[寄存器]` | 开始录制宏 |
| `q` | 停止录制 |
| `@[寄存器]` | 执行宏 |
| `@@` | 重复上次宏 |

## 九、常用配置

```vim
" ~/.vimrc 常用配置
set number                    " 显示行号
set relativenumber           " 相对行号
set tabstop=4                " Tab 宽度为 4 空格
set shiftwidth=4             " 自动缩进宽度
set expandtab                " 将 Tab 转换为空格
set autoindent               " 自动缩进
set smartindent              " 智能缩进
set cursorline               " 高亮当前行
set showmatch                " 显示匹配括号
set incsearch                " 实时搜索
set hlsearch                 " 高亮搜索结果
```

---

## 十、提示

- 大多数命令前可以加数字表示重复次数，如 `3dd` 删除 3 行
- 在可视模式下选择文本后，可以使用 `d`、`y`、`c` 等命令操作选中内容
- 使用 `:help [命令]` 可以查看详细帮助文档
